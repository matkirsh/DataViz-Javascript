Sujet Dataset

Introduction

Choix de la source et étude réalisée :

Nous avons décidé de choisir un dataset quelque peu classique pour cette étude, puisque nous avons pris sur kaggle.com un dataset nous communiquant des indicateurs sur les différents pays du monde. Néanmoins ce dataset nous offre une grande possibilité de graphique, et concentre de nombreuses informations pertinentes et comparables.

Nous nous sommes principalement attardés sur les variables mettant en valeur la taille des populations, le PIB, le taux d’alphabétisation, la présence des secteurs du service, de l’agriculture et l’industrie dans chaque pays. Ainsi que le nombre de téléphone pour 1000 habitant.

Liste des visualisations réalisées

  • Titre graphe 1 : Analyse du PIB et du taux d’alphabétisations par pays
  • Titre graphe 2 : Analyse de la présence des secteurs (industrie, service, agriculture) par continent
  • Titre graphe 3 : Analyse du nombre de téléphone pour 1000 habitant par pays

Un travail préalable a dû être réalisé sur les données, car il y avait des données manques pour certains pays, de plus il nous fallait ajouter le code ISO des pays, afin de faire matcher chaque pays présent dans notre dataset, avec les coordonnées gps des pays. Nous avons trouvé ces données dans un autre CVS, sur un github public.

Pour réaliser ce travail, nous nous sommes servis du langage de programmation Python, afin d’enlever les valeurs manquantes, de faire une analyse rapide de notre dataset dans le but, d’avoir une meilleur idée des graphiques à utiliser.
